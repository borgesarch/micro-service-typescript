### Features

* Hexagonal architecture
* Bounded Contexts
* JWT / OAUTH2
* Automatic Routes Injectors
* Automatic Entitiy Mapping
* Schedule Jobs injectors
* Notification Pattern
* Object-oriented programming
* Globalization (i18n)
