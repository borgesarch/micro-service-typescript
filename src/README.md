#  VAMOS COMEÇAR? #



### Inicio rápido com DOCKER

Em seu terminal execute:

```sh
$ docker-compose up --build
```

```sh
$ docker ps
```


### Inicio rápido sem DOCKER

Em seu terminal execute:

Buildar a aplicação build:

```sh
$ npm run ms:build
```

Iniciar a aplicação:

```sh
$ npm run ms:watch
```

Executar testes de unidade:

```sh
$ npm run ms:test
```
